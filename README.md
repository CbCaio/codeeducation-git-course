#Lista em ordem para aprender realizar o primeiro commit:

1. [Aprender o básico sobre markdown](https://help.github.com/articles/markdown-basics)
2. Realizar o commit através das instruções
  1. touch arquivo.txt
  2. vim git-commands.txt
  3. git add git-commands.txt
  4. git status
  5. vim README.md
  6. git add README.md
  7. git commit -m "Primeiro commit, arquivo contendo os comandos aprendidos do git"
  8. git push origin master
